# Classification of Corn Varieties Using Deep Learning 

**Dataset:** [Corn Image Dataset](https://www.muratkoklu.com/datasets/) — orijinal sahiplerine aittir.

## Kullanılan Yöntemler
- Custom CNN
- VGG16 Transfer Learning

## Sonuçlar
Model VGG16 yontemi ole 0.99luk basari orani yakalamistir.
